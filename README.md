# ECOMMERCE EXAMPLE

### Pasos



#### 1. Descargar Node.js y npm.

#### 2. Instalacion 

##### Clonar el repositorio 

``` git clone https://github.com/riadfrancoq/prueba.git```

##### Ir al repo 

``` cd prueba```

##### Instalar las dependecias 

``` npm install```

##### Correr el servidor

``` npm run dev```
